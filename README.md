# 🎮 Jogo 2D Platformer (Unity)

# Este projeto é um jogo de plataforma 2D desenvolvido na Unity, utilizando Tilemap e scripts em C# para controle de personagem, coleta de itens e gerenciamento básico de gameplay.

- O objetivo do projeto foi aplicar conceitos de desenvolvimento de jogos, como movimentação de personagem, física 2D, colisões, animações e sistemas de coleta.

# 🚀 Funcionalidades
Sistema de movimentação do jogador (andar e pular)
Pulo duplo (double jump)
Animações básicas (idle, correr e pular)
Sistema de Tilemap para criação de fases
Sistema de moedas/coleta de itens
Sistema de respawn do jogador
Gerenciamento básico de jogo (Game Manager e UI Manager)
# 🎮 Controles
A / D ou Setas: mover o personagem
Espaço: pular
Espaço (no ar): pulo duplo
R: reiniciar fase (se implementado)
# 🛠️ Como executar o projeto
Clone ou baixe este repositório
Abra o projeto na Unity (versão compatível com 2D URP ou padrão)
Abra a cena principal do jogo
Clique em Play para executar
# ⚠️ Problemas conhecidos
Em algumas cenas, a coleta de moedas pode não funcionar corretamente devido a colisão/trigger não configurados corretamente.
Alguns objetos podem não estar com tags ou layers corretamente atribuídos.
# 🧠 Aprendizados

Este projeto ajudou no aprendizado de:

Sistema de colisão 2D na Unity
Uso de Tilemap para construção de fases
Controle de personagem em C#
Organização de scripts em jogos
Lógica de coleta de itens e respawn
# 📌 Observação
